# Project Frontend

React + TypeScript + Vite application integrated with a Node.js backend.

## Environment Configuration

The application uses environment variables for configuration. The local environment is set in the `.env` file and managed through `src/environments/environment.ts`.

## Available Scripts

- `npm run dev`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs ESLint for code quality.
- `npm run preview`: Previews the production build locally.
